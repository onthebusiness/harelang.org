Hare is a statically typed programming language which compiles to machine code. It has assisted memory management, no runtime, and uses the System-V ABI. Hare is designed for systems programming, and is well-suited to writing operating systems, system tools, compilers, and other low-level, high-performance tasks.

Hare is considered suitable for production use today. (It will be before the public release)

Getting started

Read the installation steps to get Hare for your system, then read the tutorial. To see if Hare is available for your system, consult the supported platforms list.

Design principles

  1. Trust the programmer.
  2. But, provide tools the programmer may use when they don't trust themselves.
  3. Prefer explicit behavior over implicit behavior.
  4. Defined behavior is better than undefined behavior.
  5. In order for a program to be correct, it must be simple.
